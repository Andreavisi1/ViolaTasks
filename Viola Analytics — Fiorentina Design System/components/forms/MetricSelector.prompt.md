Segmented pill control for the ranking-chart metric switch.

```jsx
<MetricSelector value={metric} onChange={setMetric} options={[
  { value: 'goals', label: 'Gol' },
  { value: 'xg', label: 'xG' },
  { value: 'g_xg', label: 'Gol − xG' },
  { value: 'shots', label: 'Tiri' },
  { value: 'pass_pct', label: '% passaggi' },
]} />
```
